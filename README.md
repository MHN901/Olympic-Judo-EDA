# Exploratory Data Analysis: Olympic Judo 🥋📊

## 📌 Overview
This repository contains the final capstone project for the **Exploratory Data Analysis for Machine Learning** course authorized by **IBM** (Completed with a 100% score 💯).
The project merges Artificial Intelligence with domain knowledge in professional Judo to analyze historical Olympic data. 

## 🎯 Objectives
- Perform robust data cleaning and imputation on historical athlete metrics.
- Utilize domain knowledge for **Feature Engineering**: Calculating optimal Body Mass Index (BMI) baselines tailored to modern numerical Judo weight classes (e.g., -60kg, +100kg).
- Conduct statistical hypothesis testing to mathematically prove the impact of physical composition on reaching the Olympic podium.

## 🛠️ Tools & Technologies
- **Python** (Pandas, Matplotlib, Seaborn, SciPy)
- **Jupyter Notebook**
- **Data Source:** "120 Years of Olympic History" dataset from **Kaggle**.

## 💡 Key Insights
- **Engineered BMI:** Standard BMI is famously inaccurate for combat athletes as it does not distinguish between muscle mass and fat. By restricting and tailoring BMI calculations strictly to specific weight classes and separating genders, a highly precise baseline was established.
- **Statistical Significance:** A Two-Sample Independent T-test confirmed a statistically significant difference (P-value: 0.0014) in the BMI of medalists versus non-medalists, proving body composition is a critical predictor of success.

## 🚀 Next Steps
This EDA serves as Phase 1. The cleaned and engineered dataset is now optimized for Phase 2: Building Supervised Machine Learning (Regression) models to predict optimal natural weight and height profiles for athletes.
